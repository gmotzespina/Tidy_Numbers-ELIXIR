# TidyNumbers

To run the program you need to have an input file on the root of the project called input.txt with the following format:

<number_of_inputs></br>
<number1></br>
<number2></br>
.</br>
.</br>
.</br>
<numberN></br>

# Compile the project

```
mix compile
```

# Run the project

```
iex -S mix
```

# Execute

```
TN.st
```
