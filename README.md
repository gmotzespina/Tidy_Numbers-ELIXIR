# TidyNumbers

To run the program you need to have an input file on the root of the project called input.txt with the following format:

<number_of_inputs>
<number1>
<number2>
.
.
.
<numberN>

# Compile the project

```
mix compile
```

# Run the project

```
iex -S mix
```

# Execute

```
TN.st
```
